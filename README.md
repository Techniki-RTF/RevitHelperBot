## RevitHelperBot — консультант по ускорению работы с Revit‑файлами

Чат‑бот помогает инженеру ускорить работу с проектами Revit, подскажет что включить и что проверить, чтобы ускорить открытие и работу файла.

### Основные возможности
- **Пошаговая консультация**: бот проходит 8 ключевых шагов и фиксирует ответы (сделано/пропустить), в конце показывает **итоги консультации**.
- **Быстрый старт**: команда `/start`, приветствие и компактное **главное меню**.
- **Минимальный интерфейс**: вместо списка из множества кнопок, бот ведёт через диалог.

### Чеклист консультации (8 шагов)
1. **Включить Акселератор**: ускоряет открытие файла из другого офиса на 40–60%.
2. **Проверить свободное место на диске C**: нехватка места сильно замедляет Windows и Revit.
3. **Закрыть лишние вкладки/браузеры**: вкладки занимают ОЗУ — её активно использует Revit.
4. **Проверить Рабочие Наборы (РН)**: элементы должны принадлежать своим РН; минимум — РН для связанных файлов.
5. **Проверить неиспользуемые объекты**: группы, DWG, виды увеличивают вес и фоновые обновления.
6. **Контроль веса файла хранилища**: ориентир 300–500 МБ; при необходимости — пересохранение со сжатием.
7. **Аналитические данные модели**: фоновые расчёты могут замедлять; отключать по согласованию с проектировщиком/руководителем.
8. **Семейства**: использовать корпоративную библиотеку; сторонние отдавать BIM‑координатору для оптимизации.

## Установка и запуск

### Создайте файл `.env` в корне проекта со значениями окружения:

```dotenv
TOKEN=123456:ABCDEF_your_telegram_bot_token # t.me/BotFather
```
### Установите зависимости и запустите бота

```bash
pip install -r requirements.txt
python aiogram_run.py
```

## Как пользоваться
- Отправьте `/start` боту.
- Нажмите **«Начать консультацию»**.
- На каждом шаге нажимайте **«Сделано»** или **«Пропустить»**.
- В конце бот покажет краткую **сводку** и предложит вернуться в главное меню.
